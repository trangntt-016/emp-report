* [Eclipse](eclipse.org/downloads/) (install Eclipse IDE for Java EE Developers)

## Getting Started

Clone the repo
   ```sh
   git clone https://https://github.com/trangntt-016/emp-report
   cd src
   java EmployeeFrame.java
   ```
